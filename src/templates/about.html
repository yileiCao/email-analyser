Welcome to the Introduction Page

Before connecting to your Gmail account

  • Prepare your Gmail credentials by following the instructions at Google Developers.
  • Rename the credential JSON file to {{your_username}}_credentials.json and upload it to the system.

Load New Mails Page

  • You can search for and load emails from your Gmail account.
  • In addition to the commonly used fields provided on that page, you can add any tag supported by the Gmail API in the filter text column.
  • After clicking the 'Search' button, the page will display all matching emails, showing 10 emails at a time. This allows you to select which emails should be added to the database.
  • The database will automatically ignore duplicate emails.

Search Mails Page

  • You can access all emails owned by you, as well as emails shared by other users.
  • The search system supports semantic keyword search, allowing you to search for keywords with similar meanings. Semantic search only support searching with one keyword.
  • After clicking the 'Include Japanese' tag, the result set will also include keywords with similar meanings in Japanese.

View Mail Page

  • From search mails page, you will be directed to the view mail page by clicking the 'ID' at the beginning of each mail row.
  • By clicking on links in the email information table, the page will redirect to the 'Search Mails' page with that constraint.
  • You can play around with the keywords generation with different parameters or manually update keywords.
  • You can also change the sharing status of the email or delete the email from the database.
